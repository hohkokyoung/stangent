Add a user profile feature. Signed-in users can view and edit their display name (1–50 chars). They can also upload an avatar image to Supabase Storage. The screen is in the Flutter app under the existing Settings tab. The backend is FastAPI with GET/PUT /users/me endpoints. Persistence is Supabase Postgres with RLS so users can only read/update their own profile row. Avatar URL stored on the profile row.
